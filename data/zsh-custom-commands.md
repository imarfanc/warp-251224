# Common Commands (Warp)

> Tip: Use **⌘↑ / ⌘↓** to jump between command blocks, **⌘Enter** to insert/run,
> and **⌘L** to return focus to the terminal.

## Repo / navigation

```sh
pwd
```

```sh
ls -la
```

```sh
cd ..
```

```sh
open .
```

## Git

```sh
git status
```

```sh
git diff
```

```sh
git log --oneline --decorate -n 20
```

```sh
git branch -vv
```
